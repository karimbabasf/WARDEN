# M3 RADAR Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task (fresh Opus implementer + independent Opus reviewer per task). Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a live second 3D constellation ("Radar") that renders every currently-open Claude/Codex agent + subagent as globes sized by live context-window occupancy, colored by harness heat, linked parent→child — behind a top nav (Habits | Radar).

**Architecture:** A new Rust **radar collector** assembles an ephemeral live "agent forest" from local files (Claude liveness registry + `subagents/` transcripts; Codex `parent_thread_id` trees + archive-move), computes per-agent context size + honest composition, and emits a `radar_state` event. The frontend reuses the existing orb engine (layout/camera/preview/theme) behind a constellation switch, adding a depth-N radar layout, a Radar palette, lifecycle animations, and a four-section detail panel. No new heavy persistence — the forest is recomputed from files on each FS event.

**Tech Stack:** Rust (Tauri v2, rusqlite, notify/FSEvents, serde), `tiktoken-rs` (local token estimation); TypeScript + React + react-three-fiber + drei; Vitest; pnpm.

## Global Constraints

- **Package manager:** pnpm. **Platform:** macOS Apple Silicon.
- **Preview-only:** RADAR only reads local files. No writes to user projects, ever (apply = M4).
- **M4–M7 stay stubbed** via `scaffold::not_in_slice()`. Implement only `list_fleet`/`locate_agent`/`get_radar_state` for RADAR.
- **Honest viz:** estimated composition buckets are labeled "est."; flat agents (VS Code Codex, unknown harness) get NO fabricated children; every globe maps to a real signal.
- **Radar-only palette:** Claude orange, Codex violet — a separate `RADAR_PALETTE`; do NOT mutate `HARNESS` (that is Habits').
- **Smoothness:** every visual change is damped/tweened (spawn, grow/shrink, heat, implode). No snapping.
- **Verification before completion:** each task ends green — `cd src-tauri && cargo test` and/or `pnpm test`, with real output read. `pnpm build` (tsc+vite) must stay clean.
- **Commit after every task.** **Never `git push` / open a PR** without Karim's explicit instruction.
- **Frozen contract — `radar_state` payload** (Rust→web; all later tasks depend on it verbatim):
